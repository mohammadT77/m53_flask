# Maktab 53 - FrontEnd
## Task: Single-Page-Application

---
Utilize Ajax & Jquery technologies to design a Cafe landing page like the example below:  
[Cafe Template](https://www.w3schools.com/w3css/tryw3css_templates_restaurant_modal.htm)

On the server side, Implement `MenuItem` model. Then implement views below:
1. MenuItems List -> Returns a list of MenuItems **id, price, name**
2. MenuItem Detail -> Returns full detail of a menu item instance like serving_time, discount, ...
3. ContactUs -> Receives user contact form.

On the client side:
1. Retrieve the MenuItem List (1) to fill menu modal.
2. Show the MenuItem Detail (2) on click on the menu item element.
3. Use ajax to avoid page reloading while sending contact form.
